### ・関数
- 戻り値を返さない関数<br><br>
```cpp
#include <iostream>

void hello() {
    std::cout << "こんにちは！" << std::endl;
}

int main() {
    hello();
    return 0;
}
```
- 戻り値を返す関数<br><br>
```cpp
#include <iostream>

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 5);
    std::cout << "結果: " << result << std::endl;
    return 0;
}
```

### 演算子
[↑ 目次へ戻る](#toc)
#### 算術演算子
| 演算子 | 説明 | 例 | 結果 |
|--------|------|------|--------|
| `+` | 加算 | `3 + 2` | `5` |
| `-` | 減算 | `3 - 2` | `1` |
| `*` | 乗算 | `3 * 2` | `6` |
| `/` | 除算 | `3 / 2` | `1` |
| `%` | 剰余（あまり） | `7 % 3` | `1` |

<br>

### ビット演算子
| 演算子 | 説明 | 例 | 結果 |
|--------|------|------|--------|
| `&` | AND（ビットごと） | `6 & 3` | `2` |
| `\|` | OR（ビットごと） | `6 \| 3` | `7` |
| `^` | XOR（排他的 OR） | `6 ^ 3` | `5` |
| `~` | NOT（ビット反転） | `~6` | `-7` |


<br>

### シフト演算子
| 演算子 | 説明 | 例 | 結果 |
|--------|------|------|--------|
| `<<` | 左シフト | `3 << 1` | `6` |
| `>>` | 右シフト | `6 >> 1` | `3` |

<br>

### コンソールからの入力
```cpp
#include <iostream>

int main() {
    int i;    
    std::cin >> i; 
    std::cout << i; 
    return 0;
}
```
### 構造体
```
#include <iostream>
#include <string>

struct Person {
    std::string name;
    int age;
};

int main() {
    Person p;
    p.name = "Taro"; 
    p.age = 20;

    std::cout << p.name << " (" << p.age << ")" << std::endl;

    return 0;
}
```
